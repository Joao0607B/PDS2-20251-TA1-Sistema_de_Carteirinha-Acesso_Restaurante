\doxysubsection{Detailed Description}
Orquestra todas as operações do sistema de restaurante. 


\begin{DoxyItemize}
\item Gerencia os vetores de clientes e funcionários, controla o acesso, processa transações e lida com a persistência de dados em arquivos. \begin{DoxyWarning}{Warning}
O uso de \textquotesingle{}using namespace std;\textquotesingle{} em arquivos de cabeçalho é desaconselhado. Foi removido para seguir as boas práticas. 
\end{DoxyWarning}
