\doxysubsection{Detailed Description}
Representa um cliente do tipo \mbox{\hyperlink{classAluno}{Aluno}}, que possui um nível FUMP e um curso. 


\begin{DoxyItemize}
\item Esta classe herda de \mbox{\hyperlink{classCliente}{Cliente}} e ajusta o valor da refeição com base no nível socioeconômico do aluno (nível FUMP). 
\end{DoxyItemize}
