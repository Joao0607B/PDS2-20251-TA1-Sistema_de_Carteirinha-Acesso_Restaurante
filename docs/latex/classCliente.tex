\doxysubsection{Detailed Description}
Modela um cliente do restaurante, com funcionalidades de saldo, bloqueio e registro de refeições. 


\begin{DoxyItemize}
\item Esta classe herda de \mbox{\hyperlink{classPessoa}{Pessoa}} e serve como base para tipos de clientes mais específicos, como \mbox{\hyperlink{classAluno}{Aluno}} e \mbox{\hyperlink{classProfessor}{Professor}}. 
\end{DoxyItemize}
