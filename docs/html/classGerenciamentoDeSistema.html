Detailed Description

Orquestra todas as operações do sistema de restaurante.

  • Gerencia os vetores de clientes e funcionários, controla o acesso, processa transações e lida com a persistência de dados em arquivos.
    Warning
    O uso de 'using namespace std;' em arquivos de cabeçalho é desaconselhado. Foi removido para seguir as boas práticas.
