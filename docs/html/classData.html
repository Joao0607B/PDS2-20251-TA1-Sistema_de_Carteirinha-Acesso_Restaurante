Detailed Description

Modela uma data do calendário, permitindo validação e formatação.

  • A classe armazena dia, mês e ano como inteiros. Ela garante que apenas datas válidas sejam armazenadas e fornece métodos para obter a data atual, formatá-la como string e realizar comparações.
